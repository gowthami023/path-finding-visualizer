# Path Finding Visualizer

## Description
An interactive web-based tool that visualizes shortest path algorithms on a grid.

## Features
- Implemented Dijkstra, BFS, DFS, and A* algorithms
- Interactive grid with obstacle generation
- Animated path visualization

## Technologies Used
- HTML
- CSS
- JavaScript
- Data Structures and Algorithms

## How to Run
1. Download the repository
2. Open index.html in a browser
